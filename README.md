# Markov-Chains-Cha-nes-de-Markov
These are exercises that require the use of Markov chains, whether to generate graphs, identify possible options for IFOA exercises, or apply actuarial knowledge on this topic.

Add files via upload
For this, you need to have the libraries installed, primarily the `library(markovchain)`

Apple Tree FND: In this exercise, we can observe the behavior of trees in three states: healthy, unhealthy, and dead.

In addition, you can find the R file that explains how to create Markov chain states and includes some questions that may be of interest when using this model.
The second R file includes improved graphics, which is useful if you need to use it in a presentation.
